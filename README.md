# TradingView Pine Scripts

A collection of personal Pine Script indicators and strategies for TradingView.

## What is TradingView Pine Script?

Pine Script is TradingView's proprietary programming language designed for creating custom technical analysis indicators, strategies, and alerts. It's specifically built for financial market analysis and trading automation.

### Key Features:
- **Easy to Learn**: Simple syntax similar to other scripting languages
- **Powerful**: Access to extensive built-in functions for technical analysis
- **Visual**: Create custom indicators that overlay on price charts
- **Backtesting**: Test trading strategies on historical data
- **Alerts**: Set up custom notifications based on your conditions

### Pine Script Versions:
This repository uses Pine Script v5, the latest version with improved performance and capabilities.

## Available Scripts

### 1. Advisor Profit Tracker
**Location**: `scripts/advisor-profit-tracker/`

An indicator that tracks and displays profit/loss information directly on your TradingView charts.

**Features:**
- Overlays on the main price chart
- Real-time profit tracking visualization
- Built with Pine Script v5

**Usage:**
1. Navigate to the script folder
2. Copy the `.pine` file contents
3. Open TradingView Pine Editor
4. Paste and save to your chart

See the [script's README](scripts/advisor-profit-tracker/README.md) for detailed information.

## How to Use These Scripts

1. **Open TradingView**: Go to [TradingView](https://www.tradingview.com/)
2. **Access Pine Editor**: Click on "Pine Editor" at the bottom of the chart
3. **Copy Script**: Copy the content from any `.pine` file in this repository
4. **Paste & Save**: Paste into the editor and click "Save"
5. **Add to Chart**: Click "Add to Chart" to see the indicator in action
