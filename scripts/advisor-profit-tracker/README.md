# Advisor Profit Tracker

## Description
This Pine Script indicator tracks and displays profit/loss information on TradingView charts.

## Version
- Pine Script Version: 5
- Type: Indicator
- Overlay: True (displays on the main chart)

## Usage
1. Open TradingView
2. Open the Pine Editor
3. Copy the contents of `advisor-profit-tracker.pine`
4. Save and add to your chart
